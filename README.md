# EV Battery Management System (EV-BMS)

A Python-based Electric Vehicle Battery Management System designed to monitor battery performance, calculate battery health, and provide real-time visualization through an interactive dashboard.

## Features

* **Battery Monitoring**

  * Tracks battery voltage, current, temperature, and capacity.

* **State of Charge (SOC) Calculation**

  * Calculates the remaining battery charge percentage.

* **State of Health (SOH) Calculation**

  * Measures battery health based on current and original capacity.

* **Protection System**

  * Detects overvoltage and overheating conditions.

* **Cell Balancing**

  * Balances cell voltages for improved battery efficiency and lifespan.

* **Interactive Dashboard**

  * Built using Streamlit for real-time monitoring and visualization.

* **Graphical Visualization**

  * Uses Plotly for battery performance charts and battery charge gauge.

## Installation

1. Clone the repository:

git clone https://github.com/yourusername/EV-Battery-Management-System.git

2. Navigate into the project folder:

cd EV-Battery-Management-System

3. Create and activate virtual environment:

python -m venv venv

Windows:
venv\Scripts\activate

4. Install dependencies:

pip install -r requirements.txt

## Running the Project

Run the test file:

python test.py

Run the Streamlit dashboard:

streamlit run app.py

## Dashboard Preview

The dashboard includes:

* Battery metrics (Voltage, Current, SOC, SOH)
* Temperature monitoring
* Safety alerts
* Battery gauge visualization
* Cell balancing chart
